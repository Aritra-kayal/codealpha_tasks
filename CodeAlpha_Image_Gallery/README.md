# 📸 CodeAlpha Image Gallery

A responsive and interactive Image Gallery website built using **HTML, CSS, and JavaScript**. This project features a modern image slider powered by **Slider.js**, allowing users to browse images smoothly with an attractive and user-friendly interface.

## 🚀 Features

- Responsive design for all screen sizes
- Interactive image gallery layout
- Smooth image slider using Slider.js
- Clean and modern UI
- Fast and lightweight performance
- Easy navigation between images

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Slider.js

## 📸 Project Overview

This Image Gallery project was developed as part of my Front-End Development learning journey. The website displays images in a visually appealing gallery format and includes slider functionality for a better user experience.

Users can:

- View images in an organized gallery
- Navigate through images using the slider
- Enjoy a responsive experience across devices

Through this project, I learned:

- Integrating third-party JavaScript libraries
- Creating interactive user interfaces
- Improving website design and user experience
- Managing project structure effectively

## ▶️ How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Explore the image gallery and slider features.
