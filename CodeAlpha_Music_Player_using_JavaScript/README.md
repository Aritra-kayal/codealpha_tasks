# 🎵 Music Player

A simple and responsive Music Player built using **HTML, CSS, and JavaScript**. This project allows users to play, pause, and control a single audio track through a clean and modern user interface.

## 🚀 Features

- Play and Pause music
- Progress bar to track song duration
- Responsive design for all devices
- Modern and attractive UI
- Smooth user experience
- Built using pure HTML, CSS, and JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🎯 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Make sure `Ordinary.mp3` and `ordinary.jpg` are inside the `media` folder.
4. Open `index.html` in your preferred web browser.
5. Click the play button to enjoy the music.

### Clone Repository

git clone <repository-link>

Then open the project folder and run `index.html`.
