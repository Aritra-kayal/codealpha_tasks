# Simple Calculator

A simple calculator built using **HTML, CSS, and JavaScript**. This project performs basic mathematical operations with a clean and responsive user interface.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Power Function (^)
- Clear Display
- User-Friendly Interface
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Project Structure

```text
CodeAlpha_Build_a_Calculator/
│
├── index.html
├── style.css
└── README.md
```

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your preferred web browser.
4. Start performing calculations.
